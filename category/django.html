Security Features

Deployment Checklist

There is a big difference between development and production. The
settings file needs adjust when deploying a Django project.

There are many security features in Django, and some of them should
be enabled just before the deployment. A settings file for development
and a settings file for production.

The official documentation has a checklist. And a command that can
make things easier:

django-admin check --deploy
